# Arcada BSS 🐝

Arcade - is a full fledged bee swarm simulator, created based on stability, simplicity and ease of use. It combines thoughtful automation, flexible settings and a clear interface, allowing you to quickly adapt your work to your style of play.

## Features

| Module           | Description                                              |
| :--------------- | :------------------------------------------------------- |
| **Home**         | Session information and statistics                       |
| **Farming**      | Field farming, movement, conversion and collection logic |
| **Combat**       | Mobs, bosses and hostile world events                    |
| **Config**       | Flexible settings and automatic configuration saving     |
